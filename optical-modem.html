⚡ Performance

Theoretical Max:
• 1-channel (BW): 30 bps @ 30fps camera
• 3-channel (RGB): 90 bps @ 30fps
• Screen rate limited to camera fps

Practical:
• ~10-20 bps reliable
• ~1-2 bytes/second throughput
• Error rate: <5% in good conditions

Requirements:
• Bright screen facing camera
• Stable viewing angle
• Minimal ambient light changes
• Camera exposure locked if possible

Compared to:
• Early modems (300 baud): 150x slower
• Morse code: ~Same speed
• Carrier pigeon: Faster for large files

🧮 The Math

Signal Detection:
brightness = (R + G + B) / 3
threshold = (min + max) / 2 (adaptive)
bit = brightness > threshold ? 1 : 0

Clock Recovery:
Preamble 10101010 provides transitions
Period T = avg(edge_intervals)
Sample at T/2 after each edge

Checksum:
XOR all data bytes together
check = data[0] ⊕ data[1] ⊕ ... ⊕ data[n]
